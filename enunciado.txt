Debe de crear un clon del comando grep visto en sistemas linux para hallar coincidencias de patrones en archivos de texto plano.

Requisitos mínimos:

1. El programa se debe de ejecutar directamente en terminal pasándole el nombre del archivo, y la palabra a buscar coincidencias. Ejemplo:

./main.exe ./carpeta/archivo.txt palabra

2.La salida debe de ser las líneas donde se encuentran las coincidencias, además de indicar el numero de líneas con coincidencias y el numero de coincidencias.

3. Marcar en color las coincidencias.

Si se cumplen los requisitos mínimos perfectamente ya se tiene el 20. Los siguientes requisitos son para ganarse puntos extra para las siguientes evaluaciones de laboratorio o teoría, cada requisito extra cumplido es un punto extra.

Requisitos extra:

1. Encontrar coincidencias siendo parte o no de una palabra, ejemplo: si la palabra a buscar es "co", se marcaria el "co" de "como".

2. Permitir cambiar el color de la marcación de coincidencias (indicar antes de ejecutar el programa).

3. Ignorar mayúsculas y minúsculas (indicar antes de ejecutar el programa).
